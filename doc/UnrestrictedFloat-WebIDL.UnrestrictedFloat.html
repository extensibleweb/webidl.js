Class: UnrestrictedFloat

UnrestrictedFloat

new UnrestrictedFloat(value)

The unrestricted float type is a floating point numeric type that corresponds to the set of all possible single-precision 32 bit IEEE 754 floating point numbers, finite and non-finite. [IEEE-754] unrestricted float constant values in IDL are represented with float tokens.
